# AgendaVet
Plataforma de agendamento de consultas veterinárias e atendimentos de PetShop.
Desenvolvido como Projeto Integrador II do Eixo da Computação da [Univesp](https://univesp.br/)

## Resumo
O projeto desenvolvido tem como objetivo criar um sistema de agendamentos para uma clínica veterinária, utilizando tecnologias modernas como HTML, CSS, JavaScript, React, Node.js e MongoDB, além de plataformas de deploy como Vercel e Fly.io. A proposta visa facilitar o gerenciamento de consultas, proporcionando maior eficiência e organização. A metodologia adotada incluiu coleta de dados por meio de entrevistas e formulários, com uma análise mista de dados qualitativos e quantitativos. 
O grupo seguiu os passos do Design Thinking (ouvir, criar e prototipar), realizando esboços e validando o sistema junto aos proprietários da clínica. Os requisitos obrigatórios, como escolha de horário e confirmação de consultas, foram implementados no protótipo inicial, enquanto funcionalidades opcionais, como cadastro/login, múltiplas consultas e pagamento via PIX, foram sugeridas para futuras iterações. As telas do sistema foram divididas em módulos para clientes e consultório, incluindo formulários de agendamento e visualização de agenda. Como resultado, o protótipo atendeu às necessidades básicas da clínica, permitindo a organização eficaz das consultas e uma melhor gestão dos dados dos pacientes.
Considerações finais indicam que o sistema, em sua fase inicial, oferece uma solução viável e escalável para a clínica, podendo ser aprimorado com as funcionalidades opcionais, o que tornaria o serviço mais completo e flexível, tanto para os clientes quanto para os profissionais. O uso de tecnologias SaaS e infraestrutura em nuvem garantiu flexibilidade e facilidade de escalabilidade, atendendo às demandas modernas de negócios digitais
